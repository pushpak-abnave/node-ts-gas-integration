# Node.js, TypeScript & GAS Integration

## Project Overview
This project lets you write TypeScript code and deploy it for both Node.js and Google Apps Script.

## Getting Started
- Clone the repo
- Install dependencies with `npm install`
- Set up your `.env` file (see the section below)

## Building
- For Node.js: `npm run build:node`  → Outputs code in `dist/`
- For GAS: `npm run build:gas`        → Outputs code in `build/`

## Testing
- (Add `npm test` / Vitest notes if you have tests)

## Environment Variables
- Add keys like `API_KEY` to your `.env` file (never upload this file!)

## Postman API Testing
- Import `postman/api-collection.json` in Postman to use API examples.
